# Introduction #

This pages aimes to indicate the options of ports used for the motors and the sensors.

# Details #

Motor Ports:
  * Port **A**: Motor Left
  * Port **B**: _Free_
  * Port **C**: Motor Right

Sensor Ports:
  * Sensor **1**: GyroSensor Left (1)
  * Sensor **2**: ColorSensor
  * Sensor **3**: UltraSound
  * Sensor **4**: Gyrosensor Right (2)
